What is Getdata and Why Getdata


What is "Scratch-GetData"



"Scratch-GetData" is an API alternative to scratchdb. It is mainly made for scratchattach projects, and on servers that got banned by scratch, you can use this instead of scratch's API to fetch scratch data.



Why Scratch-Getdata?



Now you might ask, Why not scratchdb or some other API?" Here is your answer:

ScratchDB has lots of features, which is good, but the problem is that the accuracy is not good enough. There are some reasons you would need the data to be up-to-date, such as if you're making a live follower count.

Scratch-Getdata also has many features, and its accuracy is mostly better than scratchdb since the data is fetched directly from the official scratch API and extracted when a request is sent.

Now you may ask, then, why not just fetch from scratch instead? Well, you could, but there are servers banned from using the scratch API, so that is why @kokofixcomputers made this API so that servers banned from using scratch also get accurate information.

But for servers that are not banned from scratch, you can also fetch from here, which makes things easier and means you don't have to do all the JSON extraction and web scraping.
