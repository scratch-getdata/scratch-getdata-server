Welcome to Scratch GetData



If you use Scratch GetData please credit @kokofixcomputers on scratch



IMPORTANT: You can get the api key from the dashboard after you signup.
To use the api key you can add a key parameter like this :


https://scratch-get-data.kokoiscool.repl.co/get/user/user-exist/(username)?key=(your api key)



This is an API for Scratch, use this API to get scratch stuffs like users following count on servers banned from scratch api. This is mainly made for scratchattach backend code.


Warning: Do not use any of those and let your program go okay so this does not return something that must means the (whatever (username, etc)) does not exist for the users there is one that you can see if a user exists look for that even in projects and many more.
Disclamer: Everything you see here may be OUTDATED because this uses ScratchDB and some from Explodingstar which is more up to date



Processed Data



"Processed Data" are data that are processed so for example messages will return the number instead of the json: {"messages":"0"} making it easier for your project




Users

Note: You will have to replace (username) with the real scratch username

See if the user exist.

GET https://scratch-get-data.kokoiscool.repl.co/get/user/user-exist/(username)/
